# nadiaweb_ — Landing Page personal

Página web de presentación de servicios freelance de desarrollo web.
Construida con HTML, Tailwind CSS (CDN) y JavaScript vanilla.

## tecnologías
- HTML5 semántico
- Tailwind CSS v3 (CDN)
- JavaScript vanilla (IntersectionObserver, cursor personalizado)
- Google Fonts (Newsreader + Space Grotesk)

## estructura
index.html — archivo único con todo incluido

## deploy
Hosteado en Vercel con deploy automático desde GitHub.
Cada push a `main` actualiza la web en producción.

## para editar localmente
Abrí index.html directo en el navegador. No requiere build ni dependencias.
